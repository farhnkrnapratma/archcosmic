FArchCOS

Setup script for installing COSMIC Desktop Environment from System76 for Arch Linux and its derivatives.

# Requirements
  - git
  - base-devel
  - yay
  - chaotic-aur

# How To Get Started
  $ git clone https://github.com/farhnkrnapratma/farchcos.git
  $ cd farchcos
  $ chmod +x install.sh
  $ ./install.sh
